# Twitch Viewer

## Project Documentation

### Overview
The Twitch Viewer project allows users to view live Twitch streams and get real-time updates on their favorite channels.

### Features
- Live stream viewing
- Notifications for favorite channels
- User-friendly interface

### Installation
To install the project, clone the repository and run the following command:
```
npm install
```

### Usage
To start the application, use:
```
npm start
```
